
# Live Lunger

Live Lunger is a frontend UI/UX redesign project for a non-profit organization that promotes a healthy, smoke free lifestyle.  
The goal of this project is to create a clean, attractive, and user friendly interface that clearly communicates the mission of the organization while demonstrating modern frontend design skills.

---

##  Project Overview

This project focuses on:
- Modern UI design with React and Tailwind CSS[/
- Component-based architecture
- Clear visual hierarchy and layout\
